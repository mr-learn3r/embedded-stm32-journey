# Embedded STM32 Journey
Board: STM32F072B-DISCO (Cortex-M0, 48MHz, 16KB RAM, 128KB Flash)
Toolchain: arm-none-eabi-gcc, OpenOCD, Ceedling, VS Code

## Structure
Each week folder is added as work progresses.
- `weekX-*/` — Weekly project folders with src, notes and commit history
- `README.md` — Updated weekly with progress
